Honest numbers: the ledger records one line per delegation (time, model, agent type, description) but not token counts, so the cost index is a relative proxy — each delegation weighted by its tier's typical price ratio (editable above). Entries with no explicit model (inherit) are counted at the T4 weight, since they inherit the main-loop model — that worst-case assumption is exactly what the enforcement hook exists to prevent.

Get the hook that writes the ledger from the Enforcement and measurement section of the model-effort-router README.
